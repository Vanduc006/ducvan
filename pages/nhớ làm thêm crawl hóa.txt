          <div id="balance-steps"><h2 class="mt-3">Balance H2O = H2 + O2 Using the Algebraic Method</h2><div class="xcard mt-1 mb-3" xitemscope="" itemtype="https://schema.org/HowToX"><div class="text-center pt-2 float-xl-end mx-3"><img src="https://cdn.chemicalaid.com/assets/img/balance.php?equation=H2O+%3D+H2+%2B+O2" alt="How To Balance H2O = H2 + O2" loading="lazy" class="img-fluid" style="width:400px;height:200px;" itemprop="image"></div><div class="xcard-body mb-0"><p class="lead">To balance the equation H2O = H2 + O2 using the algebraic method step-by-step, you must have experience solving systems of linear equations.
The most common methods are substitution/elimination and linear algebra, but any similar method will work.</p>
<ol class="px-3"><li itemprop="step" itemscope itemtype="https://schema.org/HowToStepX"><link itemprop="url" href="#steps-label-vars"><h3 id="steps-label-vars" itemprop="name">Label Each Compound With a Variable</h3><div itemprop="text"><p>Label each compound (reactant or product) in the equation with a variable to represent the unknown coefficients.</p><p><span class="chemical-formula font-monospace"><span class="coefficient">a</span> H<sub>2</sub>O = <span class="coefficient">b</span> H<sub>2</sub> + <span class="coefficient">c</span> O<sub>2</sub></span></p></div></li><li itemprop="step" itemscope itemtype="https://schema.org/HowToStepX"><link itemprop="url" href="#steps-create-equations"><h3 id="steps-create-equations" itemprop="name">Create a System of Equations</h3><p itemprop="text">Create an equation for each element (H, O) where each term represents the number of atoms of the element in each reactant or product.</p><pre class="mb-0" style="overflow-x: auto;"><span class="chemical-formula"><b>H</b>:  <span class="coefficient">2</span>a     =       <span class="coefficient">2</span>b     +       <span class="text-muted">0c</span></span>
<span class="chemical-formula"><b>O</b>:        <span class="coefficient">1</span>a     =       <span class="text-muted">0b</span>      +       <span class="coefficient">2</span>c</span>

</pre></li><li itemprop="step" itemscope itemtype="https://schema.org/HowToStepX"><link itemprop="url" href="#steps-solve-vars"><h3 id="steps-solve-vars" itemprop="name">Solve For All Variables</h3><p itemprop="text">Use substitution, Gaussian elimination, or a calculator to solve for each variable.</p><div id="solve-equations-options" class="accordion card mb-3"><div class="accordion-header" id="solve-equations-options-substitution"><button class="accordion-button collapsed" data-bs-target="#solve-equations-options-substitution-content" data-bs-toggle="collapse" aria-expanded="false" aria-controls="solve-equations-options-substitution-content">Using Substitution or Elimination</button></div>

To be balanced, every element in H2O = H2 + O2 must have the same number of atoms on each side of the equation.
When using the inspection method (also known as the trial-and-error method), this principle is used to balance one element at a time until both sides are equal and the chemical equation is balanced.</p>
<h3>1. Multiply coefficients for compounds to balance out each element</h3><p>For each element that is not equal, try to balance it by adding more of it to the side with less. <ol><li><div class="chemical-formula mb-1"><b>O</b> is not balanced.
